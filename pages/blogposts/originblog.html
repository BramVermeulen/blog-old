How is this blog made?

This blog is made using bootstrap a free and open-source CSS framework, which I learned using during my studies. I decided on using bootstrap because it simplifies making a website that looks pleasing to the eye. This blog will be improved over time to make it easier to add new posts and to make it more userfriendly.

The blog is hosted by Combell on my own domain on the subdomain www.blog.bramvermeulen.be.

Why a blog?

For the course "Professional Networking", some form of a blog needed to be used to show that you keep interest in IT outside of school. This blog could be made however you wanted it just had to be accessible by an url.
