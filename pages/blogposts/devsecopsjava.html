DevSecOps Academy: Open Source Security for your Java Application

11/03/2021
Active Nexus Gitlab Java

What is DevSecOps Academy?

DevSecOps Academy is a new learningplatform which as the name sugests focusses on DevSecOps. The students that participated in the Vulnerability Scanning using SCA course got an invitation to make this course. This time we also got an introduction to the course by the DevSecOps team, who asked us if after the course we would make a last call to give feedback. Withto give feedback on this course.

Open Source Security for your Java Application

During the course used a software named Nexus lifecycle. We used this tool to scan for vulnerabilities in the open-source code. This tool similar to the debricked tool but with some sublte diffrences. This tool is also used to know if a vulnerability is severe or not and how to fix the Vulnerability. We also made this tool automaticly scan your open-source dependencies as part of your CI/CD pipeline.

Conclusion

This course was similar to the first DevSecOps course but this was intentional. The organizers wanted us to compare the two courses and give our pro and cons of each. I personally liked the debricked softare more. The course structure also felt better in the debricked one becuase there was more written theory and not all in videos. This made it easier to find the information that you needed instead of needing to go through a whole video again. I would recommend this course to every IT student that has interest in DevOps or
