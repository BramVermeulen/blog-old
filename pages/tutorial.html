Making a Discord bot using discord.py

In this tutorial you will learn how to make your own Discord bot using discord.py a python library made for interacting with the Discord API. This tutorial is focussing on how to register your own bot and how to make your won bot. Basic knowledge of Python is advised.

What will you learn in this tutorial?

  • Make a basic bot
  • Register your own Bot Account

Requirements:

  • Python 3.7.0 or greater
  • discord.py 1.7.0 or greater
  • pip 18.1 or greater

Pip install windows:

pip3 install discord.py

Pip install unix:

python3.x -m pip3 install discord.py (replace "x" with your subversion).

Make your own bot

Create a file named tutorial.py. Do not name your file discord.py as this will conflict with the discord.py library.

On the first line you write “import discord” this imports the library.

Next, you write “client = discord.Client()”. This creates an instance of a Client. This client makes it possible for us to connect to Discord.

We then write “@client.event”. This is a decorator to register an event. Because this library is asynchronous, we do things in a “callback” style manner.

A callback is a function that is called when something happens. The on_ready() event is called when the bot has finished logging in and setting things up and the on_message() event is called when the bot has received a message.

Since the on_message() event triggers for every message received, we have to make sure that we ignore messages from ourselves. We do this by checking if the Message.author is the same as the Client.user.

Afterwards, we check if the Message.content starts with '$hello'. If it does, then we send a message in the channel it was used in with 'Hello!'.

Register your own Bot Account

Login:

go to https://discord.com/login and login to your discord account. If you don't have an account click register.

Create application:

  • Go to https://discord.com/developers/applications
  • Click New Application.
  • Give your application a name.
  • Click Create.

Add a bot:

  • Click on Bot on the left side navigation.
  • Click on Add Bot.
  • Click on Yes, do it!

OAuth2:

  • Click on *OAuth2* on the left side navigation.
  • Tick the square bot inside scopes.
  • Tick the permission you would like to set for your bot. For example tick the Administrator permission.
  • Click on copy to copy the created link.

Add the bot to a server:

  • Paste the copied link into the address bar.
  • Choose a server you want to add the bot to.
  • Click Proceed.

Copy the token:

  • Go back to Bot on the left side navigation.
  • Click Copy this will give you the token for your bot.
  • Paste your token in client.run(‘’).

Conclusion

Now you have created a basic discord bot using discord.py. If you want to learn more about discord.py the discord.py documentation is linked at the end of this tutorial.
