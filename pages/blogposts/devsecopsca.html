DevSecOps Academy: Vulnerability Scanning using SCA

25/02/2021
Active debricked Gitlab Java

What is DevSecOps Academy?

DevSecOps Academy is a new learning platform, which as the name sugests focuses on DevSecOps. We got introduced to it by request of our lector Thomas Clauwaert, who asked if some students were intrested in this course. Before we started the course we had an introduction by the DevSecOps team, who asked us if after the course we would make a final call to give feedback. With the primary goal of the evaluation making improvements and tweaks to the courses.

Vulnerability Scanning using SCA

During the course used a software named debricked. We used this tool to scan for vulnerabilities in the open-source code, know if a vulnerability is severe or not and how to fix the Vulnerability. We also made this tool automaticly scan your open-source dependencies as part of your CI/CD pipeline.

Conclusion

This course was more focussed on security. Even though I am a student Sofware Engineer, this course was still has it's benefits for me. I would recommend this course to every IT student that has interest in DevOps or security.
