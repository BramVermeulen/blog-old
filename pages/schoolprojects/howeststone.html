HowestStone

What is HowestStone?

This was the first big school project I made during my bachelor in applied computer science. For this project we made a variant of the game Hearthstone that is made by Blizzard called HowestStone. This was the first time that we had to use our newly acquired skills and knowledge that we had learned in other courses in combination with one another. Because it was such a big project we learned a lot along the way.

Learning by trial-and-error

Some things did not go as planned. For example we focused to much on making the game visually appealing, while the game was not yet fully functional. This cost us a lot of time we should have allocated to functionality of the game. We also noticed during the project how important it was to ask questions if we were not certain about how we should make something.

The team

This project was a group project which I worked together on with three colleagues: Bert Van Mulders, Mattijs Step and Brend Lambert. The brother of Bert Van Mulders, Michiel Van Mulders composed a song we could use for the game. Our personal coach was Koen Cornellis.
